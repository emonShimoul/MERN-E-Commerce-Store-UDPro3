Doc for E-commerce Store:



Initials:


Step 1: Create 2 folders named backend and frontend in the root folder.



Step 2: Open the terminal in the root and install the following packages in a single line command:

*npm i express dotenv mongoose jsonwebtoken stripe cloudinary cookie-parser bcryptjs ioredis*



Step 3:Install one more dependency for development:

*npm i nodemon -D*



Step 4: Add scripts in **package.json** file for running the server and also change the type from **commonjs** to **module**.
